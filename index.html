Midi Number Tools

MIDIナンバーから音名と周波数を判定します

How to use

  1. Set Tuning frequency
  2. Set MIDI number
And, the note name and frequency are displayed.

When you press the "SOUND ON" button, a sound will be produced and a waveform will be drawn.
You can adjust the volume and drawing size by moving the volume slider.
You can also use it like a tuning fork by setting a reference sound.

  1. チューニング基準音を設定
  2. MIDIナンバーを設定
と、音名と周波数を判定し表示します。

「SOUND ON」ボタンを押すと音がでるようになり波形を描画します。
ボリュームスライダーを動かすと音量と描画サイズを調整できます。
基準音を設定することで音叉のように使うこともできます。
